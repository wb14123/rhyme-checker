# 诗词格律检测工具

检测诗词格律的命令行工具。

## 功能

* 查询单字韵律。
* 查询词牌格律。
* 根据指定词牌检测文字是否符合格律。
* 根据文字搜索最匹配的词牌。
* 支持多种韵书：平水韵，词林正韵，中华新韵。(注：平水韵为诗韵，平仄不在一个韵部中，因此对于词牌格律并不适用。)
* 集成 Claude Skills

## Build

[安装 Rust 工具链](https://rust-lang.org/tools/install/) 。

```
cargo build
```

生成可执行文件

```
./target/debug/rhyme-checker
```

## 使用

`./rhyme-checker --help` 查看使用方法。

### 与 Claude 一起使用：

直接在项目文件夹下使用 claude code，会自动读取 `.claude/skills/rhyme-checker` 中定义的内容。

效果示例：

提示词：写一首定风波描述多伦多初冬景色。

不使用此工具：

```
定风波·多伦多初冬

枫叶飘零染碧湖，
寒风渐起雁声孤。
高塔刺天云外立，
霜华密，
安大略波光如故。

初雪轻盈覆市衢，
行人匆匆各归途。
湖畔寒鸥犹独立，
凝寒碧，
冬临北国寻诗句。
```

使用此工具：

```
定风波·多伦多初冬

枫叶飘零雪欲来，湖光渐冷起风埃。
塔影朦胧霜叶败，谁怪，初冬又送雁南台。

街市萧条行客少，风早。
千帆冻泊水云开。
却喜炉边围酒暖，休叹，梅花消息待春来。
```
